offlinemot.utils_.save_tracks

offlinemot.utils_.save_tracks(tracks_objs, filename)[source]

Record the tracking results in a text file,

It saves the boxes, angels, track ids and classes after deleting the parts where no detection, tracking or movement is found, and after taking the highst class score of all the detections across the track

Parameters

  • tracks_objs (list) – The list that contains all the objects instances of the tracked tragets that should be saved

  • filename (str) – The directory of the video that is under processing. The saved text file will have the same name.
