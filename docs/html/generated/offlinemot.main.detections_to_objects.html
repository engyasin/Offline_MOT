offlinemot.main.detections_to_objects

offlinemot.main.detections_to_objects(detections, frame, last_track_id=0)[source]

Convert a list of detections out of Yolo network to traffic object class instances.

Parameters

  • frame (numpy array) – The input image in rgb format which the detections are in

  • last_track_id (int) – The id number from which numbering is starting from (default 0)

Returns

a list of the traffic object classes representing the new detections in the input

Return type

list
