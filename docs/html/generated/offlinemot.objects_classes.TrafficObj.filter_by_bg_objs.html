offlinemot.objects_classes.TrafficObj.filter_by_bg_objs

TrafficObj.filter_by_bg_objs(bg_objs)[source]

Check if the object is within a minimum distance of any moving area.

Parameters

bg_objs (list) – The list of moving object in the current frame, as regions class instances of skimage library

Returns

a tuple of two elements, where the first is boolean whether a match with one of the moving objects are found. The second is the list of moving objects after removing the matched object

Return type

tuple
